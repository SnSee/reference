
# 算法

## 正则表达式

特殊字符

```text
\      : 转义字符，将后面的字符当作普通字符而不是特殊字符
^      : 行首
$      : 行尾
*      : 匹配前面的字符 0 到任意多次
+      : 匹配前面的字符 1 到任意多次
?      : 匹配前面的字符 0 或 1 次
?      : 当该字符紧跟任何一个其他限制符（* + ? {n} {n,} {n,m}）时，匹配模式是非贪婪的（尽可能少地匹配所搜索的字符串，默认为贪婪模式）
{n}    : 匹配前面的字符 n 次（n不能是负数）
{n,}   : 匹配前面的字符 n 到任意多次
{n,m}  : 匹配前面的字符 n 到 m 次（包含m次）
.      : 匹配单个 非 \n \r 字符（[\s\S]可以匹配任意单个字符）
()     : 捕获括号中匹配内容，根据括号顺序从1开始获取，如python的 re.Pattern.group(int)
(?:)   : 不捕获匹配内容
(?=)   : 
(?<=)  : 
(?!)   : 
|      : 在单个位置指定多种匹配模式，如 x|y 可以匹配x或y，可以在()和[]中使用，如 pre(abc|123)post, pre[A-Z|1-9]post
[]     : 匹配中括号中任意字符，如 [abc] 对 a, b, c 都可以匹配，可以指定字符范围，如 [1-9] 匹配任意数字，[a-zA-Z1-9] 匹配所有英文字符和数字
[^]    : 匹配 非 中括号中任意字符，如 [^abc] 除 a, b, c 都可以匹配
\f     : 换页符
\n     : 换行符
\r     : 回车符
\t     : 制表符
\v     : 垂直制表符
\s     : 匹配任意 不可见 字符，等价于 [ \f\n\r\t\v]
\S     : 匹配任意 可见 字符，等价于 [^ \f\n\r\t\v]
\w     : 匹配任意 数字，字母，下划线，等价于 [a-zA-Z0-9_]，但可以匹配Unicode字符集中字符
\W     : 匹配任意 非 数字，字母，下划线，等价于 [^a-zA-Z0-9_]
\<     : 匹配单词开始，单词前面不能有其他 \w 字符
\>     : 匹配单词结束，单词后面不能有其他 \w 字符
```

常用

```sh
# 整数，浮点数以及科学计数法
'\d+(?:\.\d+)?(?:[eE][-+]?\d+)?'
```

特殊用法

```python
# ?: 不捕获括号中的内容
re.match(r'(?:[a-z]+)(\d+)', 'abc123').group(1)     # 输出: 123

# .*? 非贪婪模式（尽可能少的匹配，默认为贪婪模式）
re.match(r'(.*?)\d+', 'abc123').group(1)            # 输出: abc
re.match(r'(.*)\d+', 'abc123').group(1)             # 输出: abc12

# 指定多种匹配模式
re.match('pre[A-Z|1-9]post', 'preApost')            # 可以匹配
re.match('pre[A-Z|1-9]post', 'pre1post')            # 可以匹配
re.match('pre(ABC|123)post', 'preABCpost')          # 可以匹配
re.match('pre(ABC|123)post', 'pre123post')          # 可以匹配
```
