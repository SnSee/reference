
# 虚拟机

## Virtual Box

### 添加共享目录

virtual-box 设置

1. 找到共享文件夹
2. 选择要共享的 windows 文件夹，共享文件夹名称设为 share
3. 勾选自动挂载和固定分配

linux

1. sudo mkdir /share
2. sudo mount -t vboxsf share /share

### 共享系统粘贴板

常规 -> 高级 -> 共享粘贴板

```sh
# 解决 Ubuntu 中无法粘贴问题
sudo apt-get autoremove open-vm-tools
sudo apt-get install open-vm-tools-desktop
reboot
```
