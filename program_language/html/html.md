## 自动/点击跳转

```html
<head>
<meta http-equiv="refresh" content="5;url=https://www.example.com">
</head>
<body>
<p>页面将在 5 秒后自动跳转至新页面...</p>
<p>手动跳转</p>
<a href="https://www.example.com">跳转到 Example 网站</a>
</body>
```
