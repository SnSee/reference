
# python常用模块

## types

区别函数和方法(是否是static)

```python
isinstance(func, types.FunctionType)    # 普通函数，类静态函数
isinstance(func, types.MethodType)      # 普通类方法，cls类方法
```

## inspect

获取函数调用栈

```python
# 获取调用栈列表，0索引为当前函数，1索引为上一级函数
# list[inspect.FrameInfo]
inspect.stack()
```
